# LightGBM for house price forecast

## Описание
Проект обучает градиентный бустинг LightGBM для задачи регрессии —
предсказания цены дома по табличным признакам. Используется стандартный
пайплайн подготовки данных и GridSearch по гиперпараметрам.

## Небольшая теория
LightGBM — это быстрый вариант градиентного бустинга по деревьям решений.
Модель строится как сумма деревьев, каждое из которых исправляет ошибки
предыдущих. Такой подход часто дает высокое качество на табличных данных.

## Связь файлов
- `Lightgbm.ipynb` — загрузка данных, предобработка, GridSearch и обучение.
- `train.csv`, `test.csv` — данные по домам.

## Как запустить
1. Открыть `Lightgbm.ipynb` в Jupyter/VS Code.
2. Проверить пути к `train.csv` и `test.csv` внутри ноутбука.
3. Выполнить ячейки по порядку.

## Требования
- Python 3.8+ — интерпретатор Python.
- numpy — численные операции.
- pandas — работа с табличными данными.
- scikit-learn — предобработка, разбиение и метрики.
- lightgbm — градиентный бустинг по деревьям.
