# Binary search with checks

## Описание
Проект содержит реализацию двоичного поиска по отсортированному массиву.
Алгоритм возвращает индекс найденного элемента (а при наличии повторов —
индекс первого вхождения) или `None`, если элемента нет.

## Небольшая теория
Двоичный поиск работает на отсортированном массиве и на каждом шаге делит
диапазон пополам. Это дает логарифмическую сложность `O(log n)`, что намного
быстрее линейного поиска на больших массивах.

## Связь файлов
- `binary_search.py` — функция `binary_search` и набор проверок `assert`.

## Как запустить
1. Запустить `binary_search.py` в интерпретаторе Python.
2. При корректной работе все `assert` пройдут без ошибок.

## Требования
- Python 3.8+ — интерпретатор Python.
