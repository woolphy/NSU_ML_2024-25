# Housing price forecast

## Описание
Проект решает задачу регрессии: предсказание цены дома по характеристикам.
Выполнена базовая подготовка данных, после чего сравниваются несколько
ансамблевых моделей: Random Forest, XGBoost, LightGBM и CatBoost.

## Небольшая теория
- Random Forest строит много деревьев и усредняет их ответы.
- XGBoost и LightGBM — градиентный бустинг по деревьям: модели строятся
  последовательно, исправляя ошибки предыдущих.
- CatBoost умеет эффективно работать с категориальными признаками.
- В качестве метрики используется MAE (средняя абсолютная ошибка).

## Связь файлов
- `Housing_price_forecast.ipynb` — предобработка, GridSearch и сравнение Random Forest/XGBoost/LightGBM.
- `catboost.ipynb` — обучение CatBoost и формирование `catBoost.csv`.
- `train.csv`, `test.csv` — исходные данные (House Prices).
- `catBoost.csv` — пример файла предсказаний для сабмита.

## Как запустить
1. Открыть `Housing_price_forecast.ipynb` или `catboost.ipynb` в Jupyter/VS Code.
2. При необходимости обновить пути к файлам `train.csv` и `test.csv`.
3. Выполнить ячейки по порядку.

## Требования
- Python 3.8+ — интерпретатор Python.
- numpy — численные операции.
- pandas — работа с табличными данными.
- scikit-learn — предобработка, разбиение и метрики.
- xgboost — градиентный бустинг по деревьям.
- lightgbm — быстрый бустинг по деревьям.
- catboost — бустинг с поддержкой категориальных признаков.
