# Potential functions method

## Описание
Проект реализует классический алгоритм классификации — метод потенциальных
функций. Алгоритм «заряжает» обучающие объекты и затем предсказывает класс
нового объекта по суммарному влиянию соседних точек с выбранным ядром.

## Небольшая теория
Метод потенциальных функций близок к kNN, но вместо фиксированного числа
соседей суммирует вклад всех обучающих точек с весами. Вес зависит от расстояния
и ядра `K(r)`. Ядро задает, насколько быстро убывает влияние удаленных точек.
Обучение в данной реализации — это настройка «заряда» точек, которые ошибаются.

## Связь файлов
- `potential_method.ipynb` — запуск эксперимента на Iris, сравнение ядер,
  визуализация «значимых» точек и ошибок.
- `potential_funct.py` — реализация алгоритма: расстояния, предсказание, обучение.
- `kernels.py` — набор функций ядер (Newton, Tri‑cube, Gaussian, Epanechnikov).

## Как запустить
1. Открыть `potential_method.ipynb` в Jupyter/VS Code.
2. Выполнить ячейки по порядку.

## Требования
- Python 3.8+ — интерпретатор Python.
- numpy — работа с массивами и вычисления.
- matplotlib — визуализация данных и результатов.
- scikit-learn — загрузка датасета Iris и метрики.

## Резюме
Проект использовался в курсе «Введение в машинное обучение» для изучения
классических алгоритмов классификации и влияния выбора ядра.
Подходит как наглядный пример классического алгоритма классификации без
глубоких знаний ML.
