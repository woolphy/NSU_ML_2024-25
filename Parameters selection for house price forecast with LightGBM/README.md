# Parameters selection for house price forecast with LightGBM

## Описание
Проект демонстрирует, как подобрать гиперпараметры LightGBM для задачи
предсказания цены дома. Данные предварительно очищаются и кодируются,
затем модель обучается и оценивается по MAE.

## Небольшая теоретическая вставка
Градиентный бустинг строит ансамбль деревьев решений последовательно.
Подбор гиперпараметров (глубина, скорость обучения, число деревьев) помогает
найти баланс между точностью и переобучением.

## Связь файлов
- `Lightgbm.ipynb` — подготовка данных, GridSearch и обучение модели.
- `train.csv`, `test.csv` — датасет House Prices.

## Как запустить
1. Открыть `Lightgbm.ipynb` в Jupyter/VS Code.
2. При необходимости обновить пути к `train.csv` и `test.csv`.
3. Выполнить ячейки по порядку.

## Требования
- Python 3.8+ — интерпретатор Python.
- numpy — численные операции.
- pandas — работа с табличными данными.
- scikit-learn — предобработка, разбиение и метрики.
- lightgbm — градиентный бустинг по деревьям.

## Роль в дисциплине
Проект использовался в курсе «Введение в машинное обучение» как пример подбора
гиперпараметров и оценки качества модели на валидации.
Подходит как наглядный пример подбора параметров для бустинга по деревьям.
