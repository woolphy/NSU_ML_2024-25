# PR_ROC curves on simple data

## Описание
Проект показывает, как оценивать качество бинарной классификации с помощью
точности и полноты и строить PR/ROC‑кривые. Используются синтетические данные,
две базовые модели (kNN и логистическая регрессия), а кривые считаются вручную
и сравниваются с реализацией из `sklearn`.

## Небольшая теория
- `Precision` — доля верных положительных ответов среди всех положительных.
- `Recall` — доля найденных положительных среди всех истинно положительных.
- PR‑кривая строится при изменении порога вероятности; чем больше площадь под
  кривой (AUC), тем лучше модель.
- ROC‑кривая показывает зависимость TPR от FPR; AUC также характеризует качество.

## Связь файлов
- `PR_ROC.ipynb` содержит весь пайплайн: генерация данных → обучение моделей →
  вычисление метрик → построение PR/ROC‑кривых.

## Как запустить
1. Открыть `PR_ROC.ipynb` в Jupyter/VS Code.
2. Выполнить ячейки по порядку.

## Требования
- Python 3.8+ — интерпретатор Python.
- numpy — работа с массивами и вычисления.
- matplotlib — построение графиков.
- scikit-learn — генерация данных, модели и метрики.

## Резюме
В рамках курса «Введение в машинное обучение» проект использовался для отработки
оценки качества классификаторов и понимания PR/ROC‑кривых. Подходит как простое и наглядное введение
в оценку качества классификаторов.
